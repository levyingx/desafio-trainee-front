# Desafio Trainee EJECT - Theka

Projeto de desafio do processo de trainee da EJECT.

## Tecnologias utilizadas
- HTML;
- CSS;
- TypeScript;
- Vite.

## Compilação
Importe o arquivo utilizando Git:

```bash
git clone https://github.com/levyingx/desafio-trainee-front.git
cd desafio-trainee-front
```
Execute o Vite:

```
npm run dev
```
